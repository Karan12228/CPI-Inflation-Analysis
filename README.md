# Cpi-inflation

An Excel workbook created to analyze Consumer Price Index (CPI) inflation data, perform core trend calculations, and visualize the findings through charts.

## Key Features
* **Inflation Calculations:** Applied formulas to calculate inflation metrics from raw CPI figures.
* **Data Visualization:** Built charts to clearly show how inflation levels have changed over time.

## How to View the Project
1. Download the `Cpi-inflation.xlsx` file from the list above.
2. Open it in Microsoft Excel to view the calculations and charts.
